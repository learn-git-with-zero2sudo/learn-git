Cloning Repository

  1. First, we cloned the Repository from github by copying the HTTPS link under the "<> Code" tab.
  2. Second, using the terminal in MACOS or Git Bash for Windows, we need to clone the repository by using the following command: "git clone [respository link]" into the terminal.
  3. Now that you have successfully cloned the repo, you need to change the file directory of your terminal to reach the Git repo. You can do this by using the "cd" command followed by the directory of Git clone.
  4. Now you can type the "git branch" command to check what branches are available locally on your device and the main branch should show as your current branch.

Setting up your Source Tree

  1. After downloading and setting up your Source Tree account, click the "FIle" tab on the top and open the repo. If you are unsure on where to find the file, in your terminal type "pwd" to print your working directory.
  2. Once opened you will see one whole colloborative workspace which helps you visualize the process better.

Making Commits

Now you should have succesfully set up your repository clone and source tree. Open Visual Studio to make edits.

  1. You will click the "File" tab and click "Open Folder". Go to the same directory and open the learn-git repo to view the .html, .css, and .js files.
  2. Head back to your terminal and type "git checkout -b [FirstName1]". This will create a new branch with the features in main. You want to make sure you are on this branch, so type "git branch" to see the new branch and "git branch [FirstName1]" to switch to that branch. It should do it automatically when you first create the branch. This is to check for yourself.
  3. Now that you have these open make an edit to ONE of the files to keep it simple.
  4. After completing the edit you will save it and notice when you open the index.html file from your files, the changes will be made, but the changes still need to commit and push for everyone else to see.
  5. Now that you have changed one of the files you will type "git add [file you edited]" into the terminal, ex. "git add index.html". Once you do this you will commit. You commit by typing in "git commit -m '[Title in imparative sentence]'". Keep it concise.
  6. Now you will push the branch, which for first time users will require extra steps.
